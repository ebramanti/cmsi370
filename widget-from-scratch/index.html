Inspired from the Bazaar's In-Place Editor, I designed a similar widget so users of my RPG Creator have the functionality to manipulate their inputs live. In RPG, I implemented an entirely new function: when a character is selected for edit, and the edit character button is select, the current values of name, level and money are updated into a text field. The in-place text editor allows users to dynamically update their character's name, level and money values, since these can be any value (as long as "Name" is a string, and level and money are positive integers).

The most important part of this widget is that it is specifically designed for Bootstrap's built-in modal interface. The offset of the text overlay dynamically updates to the position of the modal, even when resized. This will allow users on any resolution to be able to edit character attributes within a Bootstrap modal, as long as their browser supports it.
